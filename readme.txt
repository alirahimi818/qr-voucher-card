generate QR-Code for Voucher Card. A simple and user-friendly digital Voucher card system.

== Description ==

Using this plugin, you can generate a QR-Code for the voucher and manage it.

For example, you generate a QR-Code for the desired amount for the user. The user buys from you with this QR-Code. You subtract the required amount from the voucher by scanning the QR-Code.

This plugin works with 3 shortcodes and 3 pages.

[QR_VOUCHER_GENERATE_QR] => generate QR-Code
[QR_VOUCHER_GENERATE] => make new voucher QR-Code
[QR_VOUCHER_SHOW] => show Voucher page (Voucher details...)
